# ERB Servicio Técnico

Reparación experta de electrodomésticos de gama blanca, maquinaria industrial e instalación de aires acondicionados.

**Zonas de cobertura:** Barcelona, La Coruña, Valencia, Castellón de la Plana

## Características

- Página web responsive con diseño moderno
- 3 idiomas: Español, Catalán, Valenciano
- Formulario de contacto con envío a email
- Testimonios de clientes con autenticación Google
- Google Maps con área de cobertura
- Google Analytics integrado
- Optimizado para SEO con datos estructurados (JSON-LD)

## Tecnologías

- HTML5 + CSS3 + JavaScript (ES Modules)
- Firebase (Auth + Firestore para testimonios)
- Formspree (formulario de contacto)
- Google Maps API
- Google Analytics
- Font Awesome 6
